# Archtype2K

## Technology Stack

The hospital management system was implemented using the MERN stack, which includes:

- MongoDB
- Express.js
- React
- Node.js

##  Setup Instructions
From the Root run the below commands  

```cd .\backend\```  

```npm install```  

```cd ..```  

```cd .\frontend\```  

```npm install```  

```cd ..```  

```npm i concurrently```  

```npm start```
